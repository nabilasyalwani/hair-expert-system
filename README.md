# Hair Expert System

## Cara menjalankan project

1. Clone Repo

- Clone Repo Frontend (Next.js)

```
git clone -b main https://github.com/nabilasyalwani/hair-expert-system.git
```

- Clone Repo Backend (FastAPI)

```
git clone -b backend https://github.com/nabilasyalwani/hair-expert-system.git backend-hair-expert
```

2. Masuk pada folder backend

```
cd backend-hair-expert
pip install -r requirements.txt
python app.py
```

- Server akan berjalan pada localhost port 8000

3. Buka terminal baru

4. Masuk pada folder frontend

```
cd hair-expert-system
npm install
npm run dev
```

- Client akan berjalan pada localhost port 3000
