2. Rule Base 

Rule 1: Pedikulosis Kapitis

IF (Keluhan utama = Gatal) AND (Terlihat kutu hidup OR Telur putih menempel kuat di batang rambut) THEN Diagnosis = Pedikulosis Kapitis Saran: Gunakan sisir serit dan lotion permethrin.

Rule 2: Folliculitis

IF (Keluhan utama = Gatal/Nyeri) AND (TIDAK ada kutu) AND (Terdapat bintik merah kecil OR Benjolan bernanah seperti jerawat) THEN Diagnosis = Folliculitis Saran: Hindari bercukur sementara, jaga kebersihan, gunakan krim antibiotik.

Rule 3: Seborrheic Dermatitis

IF (Keluhan utama = Gatal/Ketombe) AND (Tipe sisik = Berminyak/Kekuningan) AND (Lokasi = Kulit kepala, alis, atau area berminyak) THEN Diagnosis = Seborrheic Dermatitis Saran: Gunakan shampo ketokonazol.

Rule 4: Psoriasis

IF (Keluhan utama = Gatal/Sisik Tebal) AND (Tipe sisik = Kering, warna perak/putih, dasar merah muda) AND (Gejala penyerta = Lekukan pada kuku/Pitting OR Plak menebal) THEN Diagnosis = Psoriasis Saran: Hindari stres, gunakan pelembap urea/salisilat, konsultasi ke dermatolog.

Rule 5: Temporal Triangular Alopecia

IF (Keluhan utama = Botak sebagian/Patchy) AND (Lokasi = Pelipis/Frontotemporal) AND (Bentuk = Segitiga) AND (Durasi = Sejak kecil/Lama dan menetap) THEN Diagnosis = Temporal Triangular Alopecia Saran: Ini kondisi bawaan, opsi perbaikan adalah bedah/transplantasi.

Rule 6: Trikotilomania

IF (Keluhan utama = Botak sebagian/Patchy) AND (Kondisi rambut = Patah tidak rata/panjang berbeda-beda) AND (Perilaku = Ada dorongan mencabut rambut saat cemas/stres) THEN Diagnosis = Trikotilomania Saran: Terapi perilaku (CBT) dan manajemen stres.

Rule 7: Alopecia Areata

IF (Keluhan utama = Botak sebagian/Patchy) AND (Kondisi area botak = Licin, bulat, batas tegas) AND (Gejala khas = Ada rambut tanda seru/Exclamation point hairs OR Kuku rapuh/pitting) THEN Diagnosis = Alopecia Areata Saran: Konsultasi untuk suntik steroid intralesional jika tidak membaik dalam 6 bulan.

Rule 8: Anagen Effluvium

IF (Keluhan utama = Rontok Merata/Diffuse) AND (Kecepatan = Sangat cepat/mendadak) AND (Pemicu = Riwayat Kemoterapi, Radiasi, atau paparan Toksin baru-baru ini) THEN Diagnosis = Anagen Effluvium Saran: Rambut biasanya tumbuh kembali setelah pengobatan selesai.

Rule 9: Telogen Effluvium

IF (Keluhan utama = Rontok Merata/Diffuse) AND (Timeline = Muncul 3-4 bulan SETELAH pemicu) AND (Pemicu = Demam tinggi, Melahirkan, Stres berat, Diet ketat) THEN Diagnosis = Telogen Effluvium Saran: Perbaiki nutrisi, kurangi stres, biasanya pulih sendiri.

Rule 10: Androgenic Alopecia

IF (Keluhan utama = Rontok Merata/Menipis) AND (Progres = Lambat/Bertahun-tahun) AND (Pola = Garis rambut mundur [Pria] OR Belahan melebar [Wanita]) AND (Faktor = Ada riwayat keluarga/Genetik) THEN Diagnosis = Androgenic Alopecia Saran: Minoxidil atau Finasteride (sesuai resep dokter).
